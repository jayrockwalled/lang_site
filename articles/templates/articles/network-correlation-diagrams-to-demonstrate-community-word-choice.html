Network Correlation Diagrams to Demonstrate Community Word Choice

Visualizing Community Word Choice
Motivation

In 2015, I had been playing videogames for around 12 years. However, around February of this year when I first started following the competitive Super Smash Bros. Melee scene, was the first time I encountered a community with a completely different set of lingo than what I was used to.

The players and commentators of the game used phrases like "get bodied", "disrespect", and other much more foul language. The community seemed a lot more vulgar and raw compared to the various Nintendo communities I followed. I thought this was very interesting and took note of different community word choices whenever I was able. Eventually, I hoped to be able to quantify this.

Tools and Resources
  1. Reddit comments as the corpus torrented from this site
  2. Gensim for the NLP tools and dictionary feature
  3. Pandas for correlaton calculations and organizing data
  4. NetworkX for data visualization
Method
  1. Gather language datadata
    • Various reddit subreddit comments served as the corpus
    • I chose communities that interested me and that I thought would yield interesting results
    • Make sure to clean the data of any stop words and noise in this stage because it would save time later
  2. Build word2vec models for each community
    • Enables pulling most frequently used words
    • Useful for other various NLP analysis for other projects later
  3. NLP Analysis
    1. Within each community find the top 1000 most frequently used words
    2. Find how many of the top 1000 words intersect with each other community within each community
  4. Use Pandas to calculate correlation between each community
  5. Use NetworkX to visualize
